### 2.重重flag背后隐藏的秘密

解题步骤：根据提示写出py代码

![image-20260404003654637](C:\Users\27628\AppData\Roaming\Typora\typora-user-images\image-20260404003654637.png)

```
import re
with open(r"D:\文件\file\my_flag.txt", 'r', encoding='utf-8') as f:
    content = f.read()
pattern = r"flag{\d{3}[A-Z].*?[A-Z].\d}"
res = re.findall(pattern, content)
res1 = [i for i in res if "}flag{" not in i]
def is_valid_flag(flag):
    core = flag[5:-1]
    return (
        core[:3].isdigit()
        and core[3].isupper()
        and core[-3].isupper()
        and core[-1].isdigit()
    )
valid_flags = [f for f in res1 if is_valid_flag(f)]
print("所有匹配到的flag：", res1)
print("\n✅ 完全符合规则的flag：", valid_flags)
```

得到flag

flag值：flag{452W93bpZncSuo3JBJl3}
